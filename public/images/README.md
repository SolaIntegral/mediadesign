# 画像ファイル配置ガイド

このディレクトリには、かます結びの手順を説明する画像ファイルを配置してください。

## 必要な画像ファイル

以下の画像ファイルを配置してください：

### ステップ0: 下準備
- **ファイル名**: `step0.jpg` (または `step0.png`)
- **内容**: 紐を二重にして、二本の輪を作っている状態

### ステップ1: 交差
- **ファイル名**: `step1-1.jpg` (または `step1-1.png`)
- **内容**: 二本の輪を交差させている状態

### ステップ1: くぐらせ（重要）
- **ファイル名**: `step1-2.jpg` (または `step1-2.png`)
- **内容**: 上側の輪を下側の輪の下にくぐらせている状態

### ステップ2: 1回目のくぐらせ（重要）
- **ファイル名**: `step2.jpg` (または `step2.png`)
- **内容**: 再び上側の輪を下側の輪の下にくぐらせている状態（1回目）

### ステップ3: 2回目のくぐらせ（重要）
- **ファイル名**: `step3.jpg` (または `step3.png`)
- **内容**: もう一度上側の輪を下側の輪の下にくぐらせている状態（2回目）

### ステップ4-5: 固定＆完成
- **ファイル名**: `step4-5.jpg` (または `step4-5.png`)
- **内容**: 両端を引っ張って、結び目がロックされた完成状態

## 画像の推奨仕様

- **形式**: JPG または PNG
- **アスペクト比**: 1:1 (正方形) を推奨
- **解像度**: 最低 800x800px 以上を推奨
- **ファイルサイズ**: 各画像 500KB 以下を推奨（読み込み速度のため）

## 画像の撮影・作成のコツ

1. **真上からのアングル**: 手元の動きが分かりやすいように、真上から撮影することを推奨します
2. **背景**: 白または薄い色の背景が視認性が高いです
3. **照明**: 影が少なく、均一な照明で撮影してください
4. **焦点**: 結び目の部分に焦点を合わせてください

## 画像を追加した後の確認

画像ファイルを配置したら、ブラウザでサイトを確認してください。
現在はSVGプレースホルダーが表示されていますが、画像ファイルを追加すると自動的に表示されます。

## 注意事項

- ファイル名は正確に上記の通りにしてください（大文字小文字も含む）
- 画像ファイルが存在しない場合は、SVGプレースホルダーが表示されます
